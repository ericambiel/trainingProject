- LinearLayout
- Componentes: TextView, EditText, Button e ImageView

**App Activity:**
- Criação de Activities

- PutExtra
- Bundle

  - Button -> setOnClickListener
  - TextView -> setText
  - EditText -> getEditableText
- Java:
  - Concatenação
  - Conversão de dados: toString
  - Formatação de string: trim
